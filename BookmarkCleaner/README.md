# Chrome书签清除工具

一个安全、易用的Chrome浏览器扩展，用于批量清除浏览器书签。

## 功能特性

- 🗑️ **一键清除** - 快速删除所有书签和文件夹
- ⚠️ **多重确认** - 安全机制防止误操作
- 📊 **详细统计** - 显示将删除的书签数量
- 🎨 **美观界面** - 现代化设计，良好用户体验
- 🔒 **安全可靠** - 完善的错误处理和权限管理

## 安装使用

### 开发者模式安装
1. 下载或克隆本项目
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `BookmarkCleaner` 文件夹

### 使用方法
1. 点击浏览器工具栏中的插件图标
2. 阅读警告信息，了解操作风险
3. 点击"清除所有书签"按钮
4. 通过两次安全确认
5. 等待清除完成，查看操作结果

## 安全特性

### 多重确认机制
1. **首次确认** - 基础风险提示
2. **最终确认** - 显示具体删除数量
3. **操作保护** - 按钮禁用防止重复点击

### 权限控制
- 只在manifest.json中声明必要的`bookmarks`权限
- 严格遵守Chrome扩展安全规范
- 本地操作，不涉及网络请求

### 错误处理
- 完善的try-catch错误捕获

## 注意事项

### ⚠️ 重要警告
- 此插件会永久删除书签，操作不可撤销
- 使用前请务必备份重要书签
- 建议在测试环境中验证功能

### 性能考虑
- 大量书签删除可能需要较长时间
- 添加了操作延迟避免API限制
- 界面会显示操作进度

### 浏览器兼容性
- 支持Chrome 88及以上版本
- 基于Manifest V3开发
- 不支持旧版浏览器

## 故障排除

### 常见问题

**Q: 插件无法加载？**
A: 检查Chrome版本是否支持Manifest V3，确认文件路径正确。

**Q: 清除功能不起作用？**
A: 检查书签权限是否已授权，查看浏览器控制台错误信息。

**Q: 图标显示异常？**
A: 确认三个尺寸的PNG图标文件都存在且格式正确。

**Q: 操作过程中断？**
A: 可能是API调用限制，建议分批处理大量书签。

### 调试技巧
1. 打开Chrome开发者工具（F12）
2. 查看Console标签中的错误信息
3. 检查Network标签的网络请求
4. 使用扩展程序的"检查视图"功能
